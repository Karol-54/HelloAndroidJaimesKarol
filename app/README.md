## Respuestas

### 1. Función del AndroidManifest.xml
Es el archivo que configura la aplicación Android. Define actividades, permisos y componentes necesarios para que la app funcione.

### 2. Diferencia entre activity_main.xml y MainActivity.kt
activity_main.xml define la interfaz gráfica.  
MainActivity.kt contiene la lógica y comportamiento de la aplicación.

### 3. Gestión de recursos en Android
Android administra la memoria y los procesos cerrando o pausando aplicaciones en segundo plano para optimizar el uso de recursos.
